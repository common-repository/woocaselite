== Installation ==
1. Upload the entire woocaselite` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Woocase Lite' menu in your WordPress admin panel.

For more information have a look at the [Online Documentation](http://www.woocase.wooslide.com/help/).
